### Unit 2: Intro to HTML/CSS: Making Webpages

#### Intro to HTML

##### Welcome to the web!

The "web" is a network of connected computers. Servers serve out website data. Clients are our devices we use to browse the internet.

Websites typically utilse three languages: HTML for marking up content, CSS for styling, and JavaScript for interactivity.

##### HTML basics

The whole webpage is contained within <html> tags. Within this, there are the <head> and <body> tags.

Within the <head> tag, we have data that helps the browser render the page, but doesn't include anything the user actually sees within the page.

<meta charset="UTF-8"> is commonly included here to indicate the character set the page will render. utf-8 is commonly used as it includes the most common English characters.

You should include You may contain external CSS and JavaScript links here, or metadata, such as page title (<title>), website icon, or embedded thumbnail image.

  • <h1> - headline tag
  • <p> - paragraph
  • <br> - line break

#### Intro to CSS

##### CSS basics

<style>

    h2 {
        color: rgb(0, 232, 15);
    }  
    
    body {
        background-color: rgb(97, 250, 255);
    }
        
</style>
  • h2{} - curly brackets hold style definitions for tag <h2>, and for <body>
  • color - text colour
  • background-color - background colour
  • rgb(x,y,z) - define text colour with red/green/blue values

Many such cases! Many more styles!

##### CSS: Selecting by id

Covers assigning id's so that we can select id'd elements directly in CSS.

First, assign id in element tag like so: <p id="tag-id">

Then in the CSS <style> section of <head> we can select this id like so: #tag-id {}, adding our styles between the curly brackets.

id's are case sensitive, must not have spaces, and must be unique.

##### CSS: Selecting by class

Covers assigning classes so that we can select elements directly in CSS with class. We do this when we want to style multiple elements.

First, assign class in element tag like so: <p class="tag-id">

Then in the CSS <style> section of <head> we can select this class like so: .tag-id {}, adding our styles between the curly brackets.

Classes are also case sensitive and must not have spaces.

#### More HTML tags

##### HTML links

To link to other HTML pages, we use hyperlinks, or just links. To tag we use to do this is the anchor tag, or <a>.

We refer to the target link with a hyper reference ie. href="link.html" inside the tag.

We can also add target="_blank" to ensure the link opens in a new tab. This should be used for links targetting an outside domain.

The link text will be the text contained between the opening and closing <a> tags.

##### HTML internal links

Links can be used to navigate within a page, ie a table of contents.

We can do this using id's. Add an id to the leading element for the section you want to link to, and then when creating the link itself in your contents, define the href as that id, preceeded by a hash (#) symbol.

Define the section id:

<h1 id="project-1">
    Project 1
<h1>

Internal link to section:

<a href="#project-1">
    Internal link to Project 1
<h1>

External link to section:

<a target="_blank" href="https://website.com/index.html#project-1">
    External link to Project 1
<h1>

This is why we ensure id's are unique, as id's are typically used for functionality such as this and JavaScript.

Classes are reserved for styling.

##### HTML comments

Sometimes we want to write into our code for the benefit of ourselves or future editors. We do this with comments. These will not render on the front-end, but will be visible in the code if viewed.

To do this we use <!-- followed by our comment, and ending with -->

<!-- This is a comment -->

This is also useful for making unused content or code invisible to the user, without erasing it entirely.
